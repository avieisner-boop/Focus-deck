📱 Siri / iPhone quick capture

Set up a Shortcut once. Then say "Hey Siri, Focus Deck" from anywhere — dictate a thought — it lands in your Inbox the next time the tab is open. Optional "…and plan it" phrase opens AI Bridge with your note ready to copy.
1 — Quick capture URL
Anything Siri dictates becomes part of this URL. When the tab opens, the text auto-adds to Inbox.
2 — "Plan with AI" URL
Same idea but also opens AI Bridge, so you can copy the bundle prompt with your dictation already inline.
3 — Build the Shortcut (one-time, ~60 seconds)
  1. iPhone → open Shortcuts app → tap + top-right.
  2. Add action: Dictate Text (under Documents).
  3. Add action: URL. Paste the capture URL from step 1 above. Replace the literal text __SIRI__ at the end of the URL with a variable: tap the URL field → tap Dictated Text from the variables suggestion bar.
  4. Add action: Open URLs.
  5. Top of the shortcut: tap "Shortcut Name" → name it "Focus Deck" (whatever you say after "Hey Siri" to trigger it).
  6. Done. Test: "Hey Siri, Focus Deck" → speak your task → it'll open Safari and the task lands in Inbox.
For the "Plan with AI" flow, repeat steps with the URL from section 2 and name the Shortcut something like "Plan with Focus Deck".
4 — More Siri actions (optional shortcuts)
Build a Shortcut with a single Open URLs action (no dictation needed) for each of these. Name each one what you'll say to Siri.
5 — View this on your iPhone (Home-screen app)
  1. Open this page's URL in Safari on your iPhone:
  2. Tap the Share button (square with ↑) → Add to Home Screen.
  3. Name it Focus Deck → Add.
  4. Launch from the new icon — it opens full-screen, no Safari bars, like a native app.
Tip: Turn on Gist sync (Settings → Cloud sync) so captures from your iPhone, laptop, and anywhere else all merge together. Add to Home Screen first for the fastest open.

🤖 AI Bridge — plan with AI, import back

Hand a planning task to any AI (Claude, ChatGPT, etc.). It returns a JSON bundle. You paste it here. Matching IDs update in place; new IDs get added; nothing else is touched.
1 — Copy the prompt → paste into your AI session
Tip: paste your chat URL into the prompt before sending — the AI will attach it to every task so you can jump back.
2 — Paste the AI's bundle JSON → preview → import
Paste the entire { "format": "focusdeck-bundle", ... } object. Upsert is safe: existing items not in the bundle stay as they are.
3 — Export current state (for the AI to update)
Copy this and paste it into your AI session when you want updates rather than additions — IDs will round-trip correctly.
