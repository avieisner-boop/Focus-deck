📥 Import quick links

Two ways — pick whichever's faster on the device you're on.
Option A — Upload a Chrome bookmarks file (best on Mac/PC)
  1. Open Chrome on your laptop → click ⋮ top-right → Bookmarks & lists → Bookmark manager
  2. In the manager: click ⋮ top-right → Export bookmarks
  3. Save the HTML file (AirDrop it to your iPhone if you want to import there)
  4. Tap the button below and pick the file
Option B — Paste a list of URLs (works anywhere, incl. iPhone)
One URL per line. Optionally Label | https://url to give it a name.
— tick the ones to add to your quick links.
